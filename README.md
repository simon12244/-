# 微信多开工具

一个简单的Windows GUI工具，用于同时启动多个微信实例。

## 功能特性

- 支持一键双开、三开微信
- 支持自定义启动数量
- 自动检测微信安装路径
- 如果找不到微信，会提示用户手动选择

## 使用说明

1. 确保已安装Python 3.x
2. 下载或克隆本项目
3. 运行以下命令安装依赖：
   ```
   pip install tkinter
   ```
4. 运行程序：
   ```
   python wechat_multi_open.py
   ```
5. 在界面中选择启动数量

## 注意事项

- 使用前请关闭已运行的微信程序
- 程序会自动查找微信安装路径，如果找不到会提示您手动选择
